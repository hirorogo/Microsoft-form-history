# Microsoft Form History - 使い方ガイド

## 📝 この拡張機能について

**Microsoft Form History**は、Microsoftフォームの回答を自動保存して、いつでも履歴を確認できるChrome拡張機能です。

### こんな時に便利です
- ✅ フォームに長文を入力したのに、送信でエラーが出て内容が消えてしまった
- ✅ 過去に回答したフォームの内容を思い出したい
- ✅ 似たようなフォームで以前の回答を参考にしたい
- ✅ 入力中にブラウザが落ちて、また一から入力するのが面倒
- ✅ AI機能で効率的に回答文を作成したい

## 🚀 インストール方法

### 方法1: リリース版をダウンロード
1. [GitHub Releases](../../releases) から最新の `crx-microsoft-form-history-*.zip` をダウンロード
2. Chromeで `chrome://extensions/` を開く
3. 右上の「デベロッパーモード」をONにする
4. ダウンロードしたzipファイルを画面にドラッグ&ドロップ
5. 「拡張機能を追加」をクリック

### 方法2: ソースからビルド
```bash
git clone [リポジトリURL]
cd Microsoft-form-history
npm install
npm run build
```
その後、`dist`フォルダを拡張機能として読み込み

## 📋 基本的な使い方

### 1. フォームの回答を入力
![フォーム入力例](./screenshots/form-input.png)

1. Microsoftフォームを開きます
   - 対応サイト: `forms.office.com` または `forms.microsoft.com`
2. 普通にフォームに回答を入力します
3. **自動的に回答が保存されます**（何もする必要なし！）

### 2. AI回答提案機能を使う
![AI提案機能](./screenshots/ai-suggestion.png)

1. テキスト入力欄をクリック
2. 「AI回答提案」ボタンが表示されます
3. ボタンをクリックすると質問に応じた回答候補が表示されます
4. お好みの回答をクリックして自動入力

**AI提案の例:**
- 感想を聞かれた場合 → "大変勉強になりました"、"今後も継続的に学習したい" など
- 改善要望の場合 → "より詳細な説明があると助かります" など
- その他 → "特にございません"、"ありがとうございました" など

### 3. 保存された履歴を確認
![履歴確認](./screenshots/history-view.png)

1. ブラウザ右上の拡張機能アイコンをクリック
2. 過去の回答履歴が一覧表示されます
3. フィルター機能で表示内容を絞り込めます:
   - **すべて**: 全ての回答履歴
   - **送信済み**: 正式に送信完了した回答
   - **下書き**: 途中で保存された未送信の回答

### 4. 履歴の詳細を見る
1. 見たい履歴をクリックして展開
2. 質問と回答の内容を確認できます
3. 必須項目は赤いマーク（*）で表示されます
4. 未回答の項目は「未回答」と表示されます

### 5. フォームを再度開く
1. 履歴詳細の下部にある「📝 フォームを開く」ボタンをクリック
2. 元のMicrosoftフォームが新しいタブで開きます
3. 過去の回答を参考にして再入力できます

## 🔧 便利な機能

### 📊 送信状態の表示
- 🟢 **送信済み**: 緑色のバッジで表示
- 🟡 **下書き**: オレンジ色のバッジで表示

### 🗑️ 履歴の削除
- **個別削除**: 各履歴の「🗑️ 削除」ボタン
- **一括削除**: ヘッダーの「すべて削除」ボタン

### 📅 日時表示
- 回答した日時が分かりやすく表示されます
- 最新の回答が一番上に表示されます

## ⚠️ 注意点とトラブルシューティング

### 対応するフォーム
✅ **対応**: Microsoft Forms (Office 365、個人用)
❌ **非対応**: Google Forms、その他のフォームサービス

### こんな時は...

**Q: 拡張機能のアイコンが見当たらない**
A: ブラウザ右上の拡張機能アイコン（パズルのピース型）をクリックして、Microsoft Form Historyをピン留めしてください

**Q: 回答が保存されない**
A: 以下を確認してください:
- 対応サイト（forms.office.com、forms.microsoft.com）でフォームを開いているか
- 拡張機能が有効になっているか
- ページを再読み込みしてみる

**Q: AI提案ボタンが表示されない**
A: テキスト入力欄を一度クリックしてフォーカスしてください

**Q: 過去の履歴が見つからない**
A: フィルター設定を「すべて」に変更してみてください

**Q: フォームを開くボタンが動作しない**
A: ポップアップブロックが有効になっている可能性があります。ブラウザの設定を確認してください

## 🔒 プライバシーとセキュリティ

### 安心してお使いください
- 📱 **完全ローカル保存**: 回答データはあなたのブラウザ内にのみ保存
- 🚫 **外部送信なし**: データが外部のサーバーに送信されることはありません
- 🔐 **暗号化**: ブラウザの標準暗号化で保護されています

### 使用する権限
- `storage`: 回答データをブラウザに保存するため
- `activeTab`: 現在開いているMicrosoftフォームの情報を取得するため

## 📞 サポート

### 問題が発生した場合
1. まずブラウザを再起動してみてください
2. 拡張機能を一度無効→有効にしてみてください
3. それでも解決しない場合は、以下の情報を添えてIssueを報告してください:
   - 使用ブラウザとバージョン
   - 発生した問題の詳細
   - エラーメッセージ（あれば）

### よくある質問
**Q: 他の人に履歴が見られることはありますか？**
A: いいえ。データはあなたのブラウザ内にのみ保存され、他の人がアクセスすることはできません。

**Q: パソコンを変えた時に履歴を移行できますか？**
A: 現在のバージョンでは自動的な移行機能はありませんが、Chromeの同期機能を使用することで可能です。

**Q: 履歴の保存期間に制限はありますか？**
A: ブラウザの容量の範囲内で無制限に保存されます。手動で削除するまで残ります。

**Q: スマートフォンでも使えますか？**
A: 現在はデスクトップ版Chrome、Edge等のみ対応しています。
